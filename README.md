# 🛒 E-Commerce Application

![Angular](https://img.shields.io/badge/Angular-20-red)
![TypeScript](https://img.shields.io/badge/TypeScript-Blue)
![SSR](https://img.shields.io/badge/SSR-Enabled-success)
![Responsive](https://img.shields.io/badge/Responsive-Yes-brightgreen)

A modern and scalable E-Commerce web application built with Angular, focusing on performance, SEO, clean architecture, and user experience.

---

## 🚀 Features

### Authentication & Security

* User Registration & Login
* JWT Authentication
* Route Guards
* HTTP Interceptors

### Products

* Product Listing
* Product Details
* Categories & Brands
* Product Search
* Pagination

### Shopping Experience

* Shopping Cart
* Wishlist Management
* Real-Time Updates

### Performance & SEO

* Angular SSR (Server-Side Rendering)
* SEO Optimization
* Dynamic Meta Tags
* Lazy Loading
* Optimized Performance
* Reusable Component
* Signals

### UI/UX

* Fully Responsive Design
* Loading Indicators
* Toast Notifications
* Error Handling

---

## 🛠️ Tech Stack

### Frontend

* Angular 20
* TypeScript
* RxJS
* Angular Router
* Angular SSR

### UI Libraries

* Tailwind
* Font Awesome
* PrimeNG
* Owl Carousel
* ngx toaster

### API

* Route Academy E-Commerce API

## ⚙️ Installation

```bash
git clone https://github.com/omarmoneim/e-commerceApp.git

cd e-commerceApp

npm install

ng serve
```

Open:

```bash
http://localhost:4200
```
